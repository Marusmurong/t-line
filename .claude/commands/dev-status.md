# /dev-status — 查看工作流状态

显示当前开发工作流的完整状态信息。

## 执行步骤

1. **显示工作流状态**
   ```bash
   bash .workflow/workflow.sh status
   ```

2. **显示代理状态**
   - 列出所有已注册代理
   - 显示各代理当前任务状态

3. **显示任务进度**
   - 已完成任务数 / 总任务数
   - 当前阻塞项
   - 风险项

4. **显示质量指标**
   - 测试覆盖率
   - 代码审查状态
   - 安全扫描结果

## 用法

```
/dev-status
/dev-status --verbose  # 显示详细信息
```

## 输出示例

```
================================
  Workflow Status
================================
  当前阶段: implement
  状态:     in_progress
  启动时间: 2026-03-19T10:00:00Z
================================
  任务进度: 5/12 完成
  阻塞项:   0
  风险项:   1 (数据库迁移兼容性)
================================
```
